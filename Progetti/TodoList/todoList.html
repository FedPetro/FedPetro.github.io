Funzioni Principali

Gestione dell'input utente

L'utente potrà inserire il Task sia cliccando sul pulsante "+" che premendo il tasto Enter tramite due diversi eventi.

Aggiunta di un Task

Il Task verrà aggiunto solo se è descritto nell'input, non verrà aggiunto quindi se il campo rimane vuoto.

Eliminazione di un Task

L'utente può cancellare un Task tramite un click sul bottone "x".

Riordinamento dei Task

Tramite le frecce presenti nel Task aggiunto, l'utente ha la possbilità di spostarlo verso l'alto o verso il basso.

Task completato

L'utente una volta completato il Task può depennarlo cliccando sul bottone "check", questo porterà il task in fondo alla lista.
L'operazione è reversibile.

Struttura e Gestione degli Stati

Hooks

L'app utilizza l'hook useState per mantenere e aggiornare due stati principali:

  1. tasks: un array che memorizza la lista dei task. Ogni task è un oggetto che ha due proprietà

    • text: il testo del task.

    • completed: un booleano che indica se il task è completato.

  2. newTask: una stringa che rappresenta l'input dell'utente, ovvero il testo del task che si sta aggiungendo.
